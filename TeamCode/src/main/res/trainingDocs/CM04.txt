CM-04

DESCRIPTION:
I can import a new project in Android Studio using GitHub.

CRITERIA:
The Student knows how to import a new project from GitHub. This may happen when connecting to a
new repository, or if problems occur and the student needs to start over. The student should
understand the file structure and where to import a new project. The student should be able to
delete an old project (if necessary) and understand what happens to the project when this happens.
