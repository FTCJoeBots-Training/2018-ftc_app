CM-08

DESCRIPTION:
I can name my program files using the JoeBots standard Naming Convention.

CRITERIA:
The student understands how to name their program files. Program files should be named using the
following conventions:

Autonomous Programs should start with "auto"
Teleop Programs should start with "teleop"

Program names should be short but descriptive.

Program names should be written in "camel case" (i.e. autoRedOne).
